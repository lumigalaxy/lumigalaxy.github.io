## v0.6.3

This release updates Alien on Desk with the latest website and companion chat improvements.

### Downloads

- Windows: download the `.exe` installer.
- macOS: download the `.dmg` installer.

### Notes

- The website download button now resolves the latest GitHub Release asset for the user's platform.
- The Alien chat can bootstrap a default local Ollama model when no models are installed.
- The chat and settings UI now follow the green pixel Alien visual style.
