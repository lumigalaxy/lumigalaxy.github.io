## Alien on Desk v0.6.6

### Changed
- Replaced the visible `Alien offline` fallback with `Alien local`, so chat remains usable even when Ollama is not running.
- Switched the recommended local Ollama model to `qwen2.5:0.5b`, a small open model that is easier to pull and run locally than larger Llama defaults.
- Added chat-bubble minigames: SIGNAL for memory and COMET for quick reflex breaks.
- Added custom reminders from the alien chat bubble with minute and message controls.
- Updated the SPA-001 archive demo to use the same green flower alien visual family used by the main index experience.
