## v0.6.1

This release prepares AI Desktop Companions for public installation from GitHub Releases.

### Installers

- Windows: download the `.exe` installer.
- macOS: download the `.dmg` installer.
- Linux: download the `.AppImage` or `.deb` package.

### Notes

- Release publishing now targets `lumigalaxy/lumigalaxy.github.io`.
- Auto-update checks now read releases from the Lumigalaxy repository.
- The project README now includes simple English and Chinese installation steps for Windows and macOS.
