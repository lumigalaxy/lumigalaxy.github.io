🛸 Alien Pet · Web 嵌入组件

从桌宠项目抽出的像素外星人，纯前端零依赖，一行 import 就能嵌到你的网页里

① 用法 A：自定义元素 <alien-pet>

<script type="module" src="./alien-pet.js"></script> <alien-pet size="180" state="idle"></alien-pet>

idle（默认）

呼吸 / 眨眼 / 左右看

happy

弹跳 + 花朵旋转 + ^_^

thinking

托下巴 + 思考气泡

sleeping

慢呼吸 + Zzz 粒子

working

敲小键盘

notification

头顶金色感叹号

② 用法 B：JS API（程序化控制）

import { AlienPet } from './alien-pet.js'; const pet = new AlienPet(document.getElementById('slot'), { size: 220, messages: ['你好，地球朋友！', '欢迎来到外星人星球 🛸'], }); pet.setState('happy'); // 切换状态 pet.say('我来啦~'); // 显示气泡 pet.react('notification'); // 临时反应，几秒后自动回 idle

点击外星人身体也会触发反应 + 气泡

③ 用法 C：悬浮挂载（跟随页面角落）

import { mountFloatingAlien } from './alien-pet.js'; mountFloatingAlien({ corner: 'bottom-right', size: 160, messages: ['星球正在苏醒…', '你发现了一位旅行者'], });

召唤后会出现在浏览器窗口右下角

④ 支持的全部状态

idle · idle-follow · happy · thinking · working · sleeping · notification · error · static · mini-idle · mini-enter · mini-peek
