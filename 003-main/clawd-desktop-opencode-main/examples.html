工作原理

Clawd 是如何感知你的工作状态并做出相应反应的

事件流程

Claude Code 触发事件 → Hook 脚本处理 → 本地 HTTP 服务接收 → 状态机处理 → 展示相应动画

系统架构

基于 Electron 架构，主进程处理业务逻辑，渲染进程负责动画展示，通过 IPC 通信实现实时响应

关于 Clawd

Clawd 是一只来自 Tera-IX 星的可爱桌面宠物，专门陪伴程序员工作，让编程时光不再孤单。

基于 marciogranzotto/clawd-tank 的社区项目，由热爱编程的开发者共同维护。
