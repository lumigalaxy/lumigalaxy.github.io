# 技术架构

## 总体分层

```
┌─────────────────────────────────────────────────────────────┐
│                      展示层 (Presentation)                   │
│   apps/web (HTML+JS)         apps/desktop (Tauri+React)     │
│        │                              │                     │
│        └──────────────┬───────────────┘                     │
└───────────────────────┼─────────────────────────────────────┘
                        │
┌───────────────────────┼─────────────────────────────────────┐
│                      共享层 (Packages)                       │
│   packages/characters         packages/rag                  │
│   - 5 角色定义                - 切片 / 嵌入 / 检索          │
│   - PAD 默认值                - Prompt 拼装                 │
│   - 触发反应表                - LLM 调用客户端              │
└───────────────────────┼─────────────────────────────────────┘
                        │
┌───────────────────────┼─────────────────────────────────────┐
│                      服务层 (Edge Function)                  │
│   /api/chat (Cloudflare Workers / Vercel Edge)              │
│   - API Key 代理     - 速率限制    - 流式 SSE 转发          │
└───────────────────────┼─────────────────────────────────────┘
                        │
┌───────────────────────┼─────────────────────────────────────┐
│                      LLM 提供方                              │
│   Claude 3 Haiku  /  GPT-4o-mini  /  DeepSeek-V3            │
└─────────────────────────────────────────────────────────────┘
```

## 数据流（一次对话）

```
[用户输入]
    │
    ▼
[识别当前 species] ─── 来自 UI 状态 (data-species)
    │
    ▼
[加载角色 Bible chunks]    [加载最近 6 轮对话]    [加载当前 PAD]
    │                         │                    │
    └─────────┬───────────────┴────────────────────┘
              ▼
       [向量检索 top-3]    ── packages/rag/retrieve
              │
              ▼
       [Prompt 拼装]         ── packages/rag/prompt
              │
              ▼
       [POST /api/chat]      ── 边缘函数代理
              │
              ▼ (SSE 流)
       [流式渲染气泡]
              │
              ▼
       [解析 ΔPAD] → [更新情绪] → [触发对应动画]
              │
              ▼
       [写入 localStorage 历史]
```
