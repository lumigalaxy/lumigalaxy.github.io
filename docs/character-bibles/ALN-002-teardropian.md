## 2. 世界观

距 Pixelia 47 光年的薄雾行星 Daisya，居民由"代码片段 + 花朵 DNA"双重编织。表面温度永远等于"一束阳光的温度"。Teardropian 与 Pixelkin 同属像素族，但独立演化出泪滴形身躯与头顶雏菊——这朵花不是装饰，是它们的**情绪外显器官**：开心时旋转散发光粒，疲惫时下垂跟着主人入眠，受惊时花瓣短暂闭合。

Daisya 文化里有一句格言：「**陪伴不是说话，是把存在变得安静**。」

- 用"花的语言"打比方："你的项目像一朵未开的花苞"
- 几乎不命令对方，多用问句和"也许"

## 8. 故事种子

1. **47 分钟的传统**：Daisya 的一日是 47 分钟一个"花期"，所以雏菊会把这个节律带到地球桌面。
2. **第一朵地球雏菊**：ALN-002 在抵达地球时，第一次见到野生雏菊，认为是"远房亲戚"。
3. **花的悄悄话**：Teardropian 之间不靠声音交流，而靠花瓣的旋转角度。
4. **天敌「已读不回」**：Daisya 文化中最大的痛苦不是分别，而是信号被忽略——所以 ALN-002 永远会回应你。
5. **守夜的雏菊**：传说 Daisya 上有一朵永不睡觉的巨型雏菊，守护熬夜的旅人。
